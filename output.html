This is a standard integral in calculus. The solution is found using the following steps:

1. Rewrite tan x:

We know that $\tan x = \frac{\sin x}{\cos x}$. Therefore, the integral becomes:

$\int \frac{\sin x}{\cos x} \, dx$

2. Substitution:

Let $u = \cos x$. Then, $du = -\sin x \, dx$. This means that $\sin x \, dx = -du$. Substituting into the integral:

$\int \frac{-du}{u}$

3. Integration:

This is a simple integral of the form $\int \frac{1}{u} \, du = \ln|u| + C$, where C is the constant of integration. Therefore:

$\int \frac{-du}{u} = -\ln|u| + C$

4. Back-substitution:

Substitute back $u = \cos x$:

$-\ln|\cos x| + C$

Therefore, the solution to the integral $\int \tan x \, dx$ is $-\ln|\cos x| + C$
